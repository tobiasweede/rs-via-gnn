# RS via GNN

![image](https://cdn-images-1.medium.com/max/2400/1*AMTs2MPLKCwOZy4WaKbYTw.png)

* how to implement custom Graph Convolutional layer with _MessagePassing_

* how to build custom Graph Neural Networks
